# CoRent 베타 테스터 피드백 — 사용감 점검

_Companion to [`2026-05-06_readiness_flow_template.md`](2026-05-06_readiness_flow_template.md).
This is the **tester-facing** form. The founder hands a fresh
copy to each tester (or pastes the questions into a Google Form
/ Notion page). Do **not** show the tester the §3–§6 founder
checklist — they should react to the screens cold._

When using as a Google Form, copy questions verbatim. When using
as a written interview, the founder records verbatim Korean
quotes in §10.4 of the matching round file.

## 0. 테스터 안내문 (read aloud or paste at top of form)

> 안녕하세요. CoRent는 **사기 전에 며칠 써볼 수 있게** 도와주는
> 서비스를 만들고 있어요. 지금 보시는 화면은 **베타 단계**라
> 결제·픽업·반납·정산은 아직 연결되어 있지 않아요. 요청을
> 보내도 결제가 시작되지 않고, 셀러가 응답해도 대여가 확정된
> 것은 아니에요.
>
> 이 5분 동안 **검색 → 상세 → 요청 → 요청 목록** 순서로 한 번
> 둘러봐 주시고, 아래 질문에 솔직하게 답해 주세요. 정답은
> 없어요. "이건 무슨 말인지 모르겠다", "이건 좀 무섭다" 같은
> 반응이 가장 도움이 됩니다.

## 1. 둘러볼 화면

founder가 미리 열어드린 링크 또는 매직링크 로그인 후:

1. `/search` — 추천 카드와 `추천 이유` / `확인할 점` 영역
2. `/listings/[id]` — `구매 전 확인할 수 있는 것` 카드
3. `요청 보내기` 클릭 → 요청 성공 화면
4. `/requests` — 요청 상태 화면

(셀러 화면은 별도 라운드. 이 폼에서는 묻지 않아요.)

## 2. 질문

각 질문에 가능하면 **한두 문장**으로 답해 주세요. 이어서
"명확함 / 모호함 / 무서움 / 안심됨 / 혼란스러움" 중 가장
가까운 한 단어를 골라 주세요.

### Q1. 사용감 이해도
**`구매 전 확인할 수 있는 것` 카드를 보고, 이 물건이 어떤
사용감일지 더 잘 이해되셨나요? 어떤 항목이 가장 도움이
됐고, 어떤 항목은 굳이 없어도 됐나요?**

- 답:
- 감정 태그: ☐ 명확함 ☐ 모호함 ☐ 무서움 ☐ 안심됨 ☐ 혼란스러움

### Q2. 책임 기준 어휘
**`책임 기준`이라는 말을 봤을 때, 무섭다 / 명확하다 / 안심된다
중 어디에 가까웠나요? 그렇게 느낀 이유는요?**

- 답:
- 감정 태그: ☐ 명확함 ☐ 모호함 ☐ 무서움 ☐ 안심됨 ☐ 혼란스러움

### Q3. 결제 시작 인식 (가장 중요)
**이 페이지들을 보고 "이미 결제가 시작된 것 같다"고
느껴졌나요? 그렇게 느낀 부분이 있다면 어디였나요?**

- 답:
- 감정 태그: ☐ 명확함 ☐ 모호함 ☐ 무서움 ☐ 안심됨 ☐ 혼란스러움

### Q4. 요청 직전 멈칫
**이 물건을 요청 보내기 전에 멈칫하게 만든 건 무엇이었나요?
(없었다면 "없음"이라고 적어 주세요.)**

- 답:
- 감정 태그: ☐ 명확함 ☐ 모호함 ☐ 무서움 ☐ 안심됨 ☐ 혼란스러움

### Q5. 셀러 입장 상상
**내 물건을 등록한다고 상상해 보세요. 어떤 점이 가장
망설여졌나요?**

- 답:
- 감정 태그: ☐ 명확함 ☐ 모호함 ☐ 무서움 ☐ 안심됨 ☐ 혼란스러움

### Q6. (보너스) 단계 인식
**`아직 결제·픽업·정산은 시작되지 않아요`라는 문구를 보고,
지금이 어떤 단계라고 이해하셨나요?**

- 답:
